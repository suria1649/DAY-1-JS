window object 

window object represent the browser window 
and it is the global object in javascript.
All actions are run inside the window object. 
It is the place where everything works. 

The Window object has a number of 
properties and methods hat allows
to control the size and position of the window.



Document object 

Document object is run inside the window object.
For eg HTML is a document which is fetched inside 
window object.




Window object deals with overall outer and inner 
properties of the browser window , whereas 
document object deals with the content inside the 
window object.
